# Stuffbi Webapp Frontend

## Overview

A modern frontend application built with web technologies for the Stuffbi webapp platform.

## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

### Installation

```bash
npm install
```

### Development

```bash
npm start
```

### Build

```bash
npm run build
```

## Project Structure

```
src/
├── components/
├── pages/
├── styles/
└── utils/
```

## Technologies

- React
- JavaScript/TypeScript
- CSS/SCSS
